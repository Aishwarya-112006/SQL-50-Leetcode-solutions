# 💡 Problem Solving Approach

Instead of memorizing solutions, each problem was approached by following a structured thought process:

- Understand the database schema
- Identify relationships between tables
- Choose the appropriate SQL clauses
- Optimize joins and filtering
- Validate edge cases
- Keep queries clean and readable

This repository intentionally emphasizes understanding over shortcuts.
